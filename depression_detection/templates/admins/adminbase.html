A Deep Learning Library for Anomaly Detection

This open-source library provides algorithms from the literature and a set of tools to design custom anomaly detection algorithms via a plug-and-play approach.

introducing

depression_detection is an extensive library for the design, implementation, and deployment of unsupervised anomaly detection models from data to the edge.
